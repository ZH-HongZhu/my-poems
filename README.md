# 醒世问天
**——赤筠 作**

---

### 题记
> 盖闻华夏五千载，治乱兴替，系乎生民。自上古教民稼穑，辟土植桑，民生始立，邦本始固。三代以降，皇权既立，神权渐隐，民始由“人”沦为“臣”。
>
> 秦汉以还，门阀迭起，礼教相循，阶级如铁；虽有陈胜之呼、黄巢之怒，誓欲踏平豪门，然终未破千年之网。直至近代，教员振臂，开天辟地，方将“人民”二字，由草芥升格为江山之主。
>
> 时至今日，物阜民康，远胜往昔。虽有世事繁杂、利弊相生，然历史向前，终是正道。
> 然余以冷静之逻辑，静观今世，心有所忧：
> 所忧非一时之困，乃生养渐稀、族脉渐微；
> 所叹非一己之遇，乃城乡离散、烟火渐稀。
>
> 国以人为本，人以育为续。无民则无土，无嗣则无传。华夏文脉，非赖金帛，而赖生生不息之子孙。
>
> 故作此篇，实为醒心。
> 愿世道知所重，生民知所续，
> 守我山河，延我文脉，续华夏之传！

天下趋利，驱民如役；
器进本为济苍生，资本缚之尽作羁。
财环流衍，举世尽羁绁。
操柄者坐收回流之利，众生碌碌，皆为供养。
 
学历为藩筛。
筛苍生、分等定役，尽成耗材。
外包如芥、派遣如隶，任劳倍常而利半，
综合包干命如磬，身心俱瘁损形神，生机渐噬。
名虽灵活、实是囚役，积怨微萌便弹压，
池金灭火压众声。维稳为先，对错全抛却。
 
背锅顶罪、皆归此辈。
工牌分色、岗位分席，福利全剔、晋升无梯。
工伤责相踢，微庇难遮老病身。
名为用工、实是赁役，终身难脱籍。
 
体制内、皆棋格。
上下俱为缰锁役，利链相牵、非独私门客。
货币刃，割民脉。
币值渐蚀，购买力消，厚利分流、债负沉压，终以民生承之。
 
利逐回流重短策，稳敛厚金则。
纵政策、全倾此辙。
未扼资本侵生业，先丘墟后黎民骨。
官廪凭流转续，俸薪支度相系。
三者缠死成一体，众生尽在羁绁。
本末倒置，金役而已。
池涸水穷，仍掘来日。
生产虚盈，创新多滞，利心常驱，惟逐利息。
本末乱，人如寄。
 
民生尽作圈中利，市井生存皆成贾。
财补民补乍施泽，转瞬供给套取，闭环成壑。
表演恩慈空抚慰，诺重行赊无着。
一裁两审磨人志，调解拖残心力。
法条成框，执行滞后，依例惯性，人本沉陆。
纵维权、价重身竭。
 
万般努力终归幻，脱颖英才能几人？
拼至身残魂亦碎，几多血汗葬风尘。
 
积弊挤压生存地，底层互害相煎急。
卷工时、竞低价，自相残戮。
资本虚浮离实业，只逐金利空轮转。
平台为链、数据为鞭，算法驱役榨残力，都市白蓝领，尽入盘剥局。
平台抽成、流量为刃，线上亦成收割地。
教人致富皆圈套，全链收割无遗力。
加盟锁、料器缚、品牌缚，房租更作千斤枷。
纵是个体营生者，亦作资本羁役。
 
巷口摊贩、街边走卒，晨兴奔生计、带月守残摊。
城管来、便收篷，城管去、复开张。
租金逐、换地方，方寸之地巧周旋。
苛禁稍弛非施惠，实是尘泥榨已干。
明宽暗紧成软缚，生计犹作釜中煎。
薄利皆从血汗出，千折百挠不断烟。
微末生涯凭韧性，风雨之中犹自立。
 
村头老幼倚门望，数尽晨昏盼归期。
稚子空呼爹娘远，白头独对红灯泣。
经年一别难相聚，满心思盼总成痴。
亲情割裂谁相问，留守辛酸人不知。
游子身羁都市牢，欲接双亲力不逮。
欲归乡土子难从，学籍户籍锁尘埃。
三代离散各天涯，乡镇渐空人不在。
唯有荒村留守老，空对寒门等儿来。
 
漫道增存图扩饼，体制早定格局。
饼广只肥分羹客，于我何曾有滴沥？
 
虽有转型高新技，尽作权门嫁衣。
英才枉作登云梯，更以情怀相羁縻。
微恩小惠虚安抚，难掩釜底沸鼎。
钝刃迟磨生趣尽，磨到魂枯族弱尽残孑。
 
自以为醒，皆在羁绁。
积弊根深难自破，内有鸣者皆摧折。
铁板一盘同舟系，万绪千丝缚国脉。
技术趋固沉疴重，变革须趁今朝计，莫待机失万丈崖。
 
更忧流俗移根本，前车覆辙、文脉渐改。
劳役趋利补空额，客籍渐聚，旧泽犹存，世风日移。
社保承压，寅吃卯粮，资本驱役。
预铺前辙，凭惯性，暗为来日。
惠政虽施难救弊，沉疴岂独补贴平？
 
寰球同此殷忧，联合国亦为之警：
生齿日稀，文脉将替，乡土日空，邦本渐摇。
此非一人之私虑，实乃天下之公忧。
 
西洲巧殖无兵革，金链为囚，暗噬神州脉。
内有伥魁为媒介，引寇分肥，卖却苍生社稷。
外邦吮血凭规则，内蠹盘根成羽翼，
共筑樊笼缠禹甸，生生元气渐萧森。
 
城无安处，乡无归途，底层无产皆依附；
非甘驰驱离乡土，实因生计不容身。
 
风过荒村，草动如泣；
月照空城，影孤似诉。
 
最是乡农承国重，昔年粒米济邦家。
税赋工役填基壤，血沃田畴筑华夏。
今作民工奔市井，路桥广厦亲手建。
广厦半成枯骨立，一生积蓄付东流。
世情偏薄轻相待，社保惟缴最低档。
握契空称地主名，⼨土难耕唯出租。
资本下乡吞阡陌，年租不及一月工。
辛劳倾尽儿女债，半生积储付首付。
燃尽余生无归处，徒留空契守荒篱。
更怜子弟难越阶，资源锁死在尘泥。
城乡沟壑深如堑，世代沉浮底层役。
 
宅舍虚价无造血，徒系环流、何曾是资储。
足踏实地，心若浮萍，根脉早被利流断。
层僚渐与黎民隔，中下相疑，矛盾暗结。
 
当道多尸禄，惩贪只扫浮尘，根基何曾撼。
但求权位无倾塌；家族可遁，资产可移，子孙远别此疆土。
更欲引、外客填阙，
算来治理、无根最易驱役。
无史无归无乡念，俯首惟知顺役。
较吾辈、青年难制。
有骨有魂有家邦，偏生难驭，反成嫌忌。
 
空诺何曾安社稷，系统行来终是劫。
忍看华夏衣冠熄，
纵教神州沦异类，当事者、何曾珍惜？
只图一身安且逸，抛却千秋宗祀。
 
五十六族千年合，血沃中原、汗耕阡陌。
基建完、为谁而设？岗位虚、为谁而留？民生五事成外泽。
智能机器难尽代，偏用远人、轻弃家国。
福利偏倾天外客，我辈寻常、反成卑弱。
婚嫁尽作阶层契，竞逐编内求安稳，深缚环流稳依托。
榨尽余价成刍狗，驱入尘途竞奔忙。
外卖驰街、网车疲影，直播谋生、公考研场。
千军踏破求生路，场畔虚悬，慢就业字彰。
 
莫道围城即乐土，金漆牌匾下，亦是困兽斗。
案牍劳形，文山会海，青丝熬成雪，初心付东流。
层级如梯，人情似网，谨小慎微，步步费思量。
昔日屠龙技，今作雕虫术，棱角磨平处，方知笼中囚。
上岸非彼岸，此岸亦荒凉，同是红尘客，谁解其中愁？
 
休言登堂便脱俗，不过是，高级耗材入册。
纵有朱批权在手，终是，齿轮咬合处，无声消磨。
层级森严，品阶错落，低中高下，皆是工具格。
你以为的青云路，实则是更精致的枷锁。
系统运转，何曾问，工具悲欢？只论功用与取舍。
待到油尽灯枯日，谁管你，是金漆，还是铁索？
同是人间负重身，何来高低贵贱说？
 
岗分泾渭判云泥，官吏企在编，方属恒产真资储；
体制器用分等次，底层耗材也有别；
民营外聘皆依附，临雇派遣尽羁役，倾轧相争苦。
万头攒动夺一席，互踩互卷，风骨尽作尘与土。
系统运转循私利，尽以苍生为养分，一路行来负苍生。
 
资本本不事生产，亦未劳身付心力。
物与生灵皆载体，货币为器人作役。
罔顾劳作本为人，反教苍生困钱羁。
 
逐利初惟速流转，资本先吞地、复噬形骸。
租尽丘墟租尽命，苍生都成赁。
古为佃农今房奴，一身辛苦皆为租。
教婚医养全栓缚，寸土锁尽此生途。
 
读书役、营生役、婚娶役、房债役、风险役，
一生追求总成空，枉自为人。
心放平、身少缚，
学历金帛屋舍，皆是浮尘外物。
惟康健身心，方是真珍。
 
系统早被资本缚，唯效唯利、人本沉陆。
惯性滔滔难自返，不刹危车终临深谷。
 
士登据要津，后进跻攀艰。
守心遭倾轧，从利役生民。
 
华夏薪火相传久，岂容断自今人之手？
我辈先贤，皆曾力挽狂澜、惊绝当世。
谁忆砥柱定乾坤辈，当年血肉筑家国。
 
到此际、噬魂彻骨。
先祖热血浇中土，竟换贪佞苟安易。
山河泪，无人祭。
同抱丹心守文脉，立身传家、不教根绝——
也待清风，更誓争一线。

乱曰：
收回民市，以民惠民。
休养群生，复我乾坤。
